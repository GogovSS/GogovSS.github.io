Презентация по HAML

HAML - инструмент для упрощения html- вёрстки.

Вставка html-кода

Каждый тег создаётся с помощью %название_тега (например %p). Всё содержимое тега должно выделятся отступом. Тег заканчивается там, где заканчивается отступ.
Для присвоения тегу class или id после названия тега нужно написать .имя_класса или #идентифекатор. Для присвоения для аргументов нужно ещё написать (название_атрибута = значение_атрибута). 
					
%html
	%head
		%title HAML
	%body
  	%h1.head Haml программа
		%p
	    #text HAML - инструмент для упрощения html-вёрстки.
	    %a(href = “https://haml.ru/”) о HAML

Вставка CSS и JS

Для того, чтобы вставить CSS код (подключить стили из CSS-файла), напишите
%link(rel = "stylesheet", href = "имя_CSS_файла" )
С JavaScript аналогично - напишите
%script(src = "имя_JS_файла")
